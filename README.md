# Yandex Search UI Tests

Simple UI automation tests for Yandex search using:

- Java
- Selenide
- JUnit 5
- Maven
- Page Object pattern

## Test cases

- searchInYandex
- searchInYandexEnglish

## Project structure

src/test/java/pages/YandexPage.java  
src/test/java/tests/YandexSearchTest.java

## Run tests

mvn test
